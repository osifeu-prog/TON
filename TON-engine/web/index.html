SLHNET · TON Analytics · Botshop · SLH Wallet

זוהי סביבת המסחר, הניתוח והקהילה המלאה שלך – מנוע ניתוח טכני חי ל-TON/BNB, גישת בוט כניסה (Botshop) וארנק SLH מלא.

לוח מצב · TON Engine (Flask)

מציג את סטטוס השרת והחיבורים.


        


        

            
ניתוח טכני חי

            

                שימוש בפונקציות הניתוח הטכני המתקדמות שנמצאות במנוע:
                RSI, MACD, Bollinger Bands, ADX, ATR ועוד.
            

            
            
            

        


        

            
ארנק קהילה · SLH Wallet

            

                שכבת הארנק (slh-wallet) מספקת API ליתרות, תנועות ומדדים פיננסיים של קופת הקהילה.
                כאן תוכל לבדוק את היתרות כפי שהן מדווחות מהשירות החי.
            

            

                
            

            

                ברירת המחדל מניחה שהשירות רץ על http://localhost:8081 עם נקודת קצה /balances.

                ניתן לערוך את כתובת השרת בקובץ web/app.js תחת הקבוע WALLET_BASE_URL.
            

            

        


        

            
SLH Ecosystem Integration

            

                המערכת הזו נבנתה כשכבה טכנית בתוך האקו־סיסטם הרחב של SLH.
                המבנה המלא במונורפו:
            

            
    
                
  • TON-engine/ – מנוע הניתוח והסיכונים (Flask + ניתוח טכני).
    • 
                
  • botshop/ – שער כניסה / בוט אקו־סיסטם SLH (קמפיינים, לידים, קבוצות).
    • 
                
  • slh-wallet/ – שירות ארנק SLH / API לשכבת הבלוקצ׳יין והמעקב הפיננסי.
    • 
            

            

                לצורך הרצה מקומית, ניתן להפעיל את ה-TON Engine כמרכז הניתוח ואת הבוטים/ארנקים
                על פי ההוראות בתיקיות botshop/ ו-slh-wallet/.
            

        


        

            
Botshop · שער הכניסה לקהילה

            

                Botshop הוא בוט/שירות כניסה שמנהל לידים, הצטרפות לקבוצות, וקמפיינים לקהילת SLHNET.
                ניתן לחבר אותו לאתר ולמנוע הניתוח על ידי קישורים ו-Webhook.
            

            

                ברמת הקוד, Botshop נמצא בתיקייה botshop/ במונורפו, וכולל:
            

            
    
                
  • בוט טלגרם שמטפל ברישום והפניות.
    • 
                
  • דפי נחיתה (templates/static) לשיווק.
    • 
                
  • תיעוד מלא בפרק docs/ של הפרויקט.
    • 
            

            

                אפשר להוסיף כאן קישור ישיר לבוט הטלגרם שלך (למשל https://t.me/YourBot) או לשרת production של Botshop.
