# ארכיטקטורת SLH FULL SUITE

המונורפו מכיל שלושה שירותים עצמאיים שעובדים יחד:

- `TON-engine/` – מנוע ניתוח טכני, Flask API, שכבת סיכונים ודשבורד + אתר Web.
- `botshop/` – בוט/שירות כניסה לקהילה, לידים, קמפיינים ותיעוד מלא.
- `slh-wallet/` – שירות ארנק SLH / API פיננסי (FastAPI/Telegram), כולל DB ומטריקות.

פירוט שכבות, קשרים בין השירותים ודוגמאות אינטגרציה מפורטים בתיעוד השיחות שלנו, ויכולים לשמש כבסיס להמשך פיתוח.
